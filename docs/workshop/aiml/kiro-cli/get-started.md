## 1.Get Started / 시작하기

Kiro CLI는 터미널 환경에서 AI 기반 개발 지원(AI-assisted Development) 기능을 제공하는 도구로, 자연어 명령과 자동화된 워크플로우를 활용해 애플리케이션을 빌드(Build), 테스트(Test), 배포(Deploy)할 수 있도록 해줍니다.

이 페이지에서는 Kiro CLI를 가장 빠르게 시작하는 방법을 안내합니다.

***

### 1.1. Kiro CLI 설치(Installation)

macOS 및 Linux 환경에서는 아래 명령어 한 줄로 Kiro CLI를 설치할 수 있습니다.

```
curl -fsSL https://cli.kiro.dev/install | bash
```

설치가 완료되면 새로운 터미널 창을 열고 다음 단계로 진행합니다.

***

### 1.2. Kiro CLI 실행(Start Using Kiro CLI)

프로젝트 디렉터리로 이동하여 Kiro CLI를 실행합니다.

```
cd my-project
kiro-cli
```

정상적으로 실행되면 터미널에서 Kiro와 직접 상호작용하며 개발을 진행할 수 있습니다.

***

## 2. Use Cases / 활용 사례

Kiro CLI는 다음과 같은 상황에서 특히 효과적입니다.

#### • 인터랙티브 개발(Interactive Development)

터미널에서 Kiro와 직접 대화하며 코드 생성, 문제 해결, 분석 등 즉각적인 도움을 받을 수 있습니다.

#### • 맞춤 자동화(Custom Automation)

특정 워크플로우에 최적화된 전용 에이전트(Specialized Agents)\*를 생성해 반복 작업을 자동화할 수 있습니다.

#### • 팀 표준화(Team Standardization)

팀 차원의 베스트 프랙티스(Best Practices)와 공통 설정을 반영하여 일관된 개발 환경을 구축할 수 있습니다.

#### • 외부 서비스 통합(External Integrations)

MCP(Manifold Control Protocol) 서버를 통해 외부 도구 및 서비스와 Kiro CLI를 연결할 수 있습니다.

#### • 지능형 지원(Intelligent Assistance)

컨텍스트(Context)를 이해하는 AI 기반 제안, 자동 완성, 코딩 가이드 등을 통해 개발 효율을 높일 수 있습니다.

#### • 워크플로우 최적화(Workflow Optimization)

스마트 훅(Smart Hooks)\*을 사용하여 반복적이거나 규칙적인 작업을 자동화할 수 있습니다.

### 요약(Summary)

* Kiro CLI는 AI 기반 개발을 터미널에서 직접 사용할 수 있는 도구입니다.
* `curl ... | bash` 명령으로 간단히 설치할 수 있습니다.
* kiro-cli 실행 후 즉시 Kiro와 상호작용하며 개발을 시작할 수 있습니다.
* 다양한 자동화·통합·지능형 지원 기능을 통해 개발 생산성을 크게 높일 수 있습니다.
